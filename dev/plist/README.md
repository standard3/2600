# PList

## Description

PList (Process List) is a CLI tool that prints current running processes details and their associated threads on the local computer.

## Build

Build the binary :

```powershell
$ mingw32-make all
or
$ make all
```

## Usage

```powershell
$ .\bin\plist.exe -h
Usage: plist.exe [-h|-d] <process>|<pid>
```
